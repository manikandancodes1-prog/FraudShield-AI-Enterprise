## 📌 Overview

FraudShield AI is a **real-time enterprise fraud detection platform** built using modern full-stack technologies.

It monitors transactions, calculates AI-driven risk scores, and provides live fraud alerts using WebSockets.

Designed with scalable microservice architecture and containerized using Docker.

---

## 🧠 Key Features

✨ Real-time transaction monitoring  
📊 AI-based risk scoring engine  
🚨 Live fraud alert notifications  
🔐 JWT-based authentication  
📈 Interactive analytics dashboard  
📦 Dockerized multi-container architecture  
⚡ Redis-powered streaming support  
📑 Exportable fraud reports  

---

## 🏗️ Architecture

Frontend → FastAPI Backend → Redis → PostgreSQL
